# Villes du Futures 

Pour accèder à la page d'acceuil du site veuillez cliquez sur cette url : https://mathurus.github.io/index.html

 Rappel des règles du jeux : 

 Après avoir choisi un pseudo et le nom de votre ville, vous allez devoir choisir des matériels, vous pourrez ensuite l'améliorer, et passer les jours pour gagner de l'argent et des scores. 

 Le but étant d'avoir 0 de pollution, 100 d'énergie, 100 de satisfactions.
